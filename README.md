### 커서를 한 단어 왼쪽으로 이동  
Key : ⌘ + ←  
Action : 0x1b 0x62  

<br>

### 커서를 한 단어 오른쪽으로 이동  
Key : ⌘ + →  
Action : 0x1b 0x66  

<br>

### 커서를 줄의 시작으로 이동  
Key : ⌘ + ⇧ + ←  
Action : 0x01  

<br>

### 커서를 줄의 끝으로 이동  
Key : ⌘ + ⇧ + →  
Action : 0x05  

<br>

### 단어 삭제  
Key : ⌘ + ←Delete  
Action : 0x1b 0x08  

<br>

### 줄 삭제  
Key : ⌘ + ⇧ + ←Delete  
Action : 0x15  

<br>
